Damn Vulnerable Web Application

Aplikacja internetowa stworzona specjalnie w celu szkolenia w dziedzinie testów penetracyjnych i zrozumienia zagrożeń związanych z bezpieczeństwem aplikacji internetowych.
DVWA jest stworzone w języku PHP i używa bazy danych MySQL. Głównym celem DVWA jest dostarczenie platformy do praktycznego uczenia się i testowania umiejętności związanych z bezpieczeństwem aplikacji internetowych, takich jak ataki SQL injection,
Cross-Site Scripting (XSS), Cross-Site Request Forgery (CSRF) itp.


CSRF

Cross-Site Request Forgery (CSRF) to atak, w którym atakujący wykorzystuje uprawnienia zalogowanego użytkownika do wykonania niechcianych działań na innej stronie internetowej, na której użytkownik również jest zalogowany.

Zmiana typów inputu 5. Modyfikacja przycisku submit 6.Dopisanie value='root' do pola formularza, w celu ułatwienia użytkownikom testowania różnych zagrożeń związanych z hasłami 7.
